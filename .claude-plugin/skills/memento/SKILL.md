# Memento

Memento runs as an MCP server (`memento-mcp`) and exposes tools for:
- Memory add/search/list
- Active Coercion rules (code immune system)
- Goal alignment checks (Strict Mentor)
- Dependency audits (Dependency Tracker)

Use the MCP tools directly (preferred), or consult the repository README for setup and examples.
